# WeatherView

Developed by: Daniel Delgado, John Shanahan, and Jadon VanSchaick

WeatherView is an iOS Weather app which allows users to input a location to retrieve weather data for, and displays the results in the UI. Users can view the current weather, 24 hours of future weather, and 7 days of weather at-a-glance. Users can also choose between Metric and Imperial for their display.

All of these choices are saved and persistent across multiple sessions of using the app!

## Installation Instructions

To install and run WeatherView, you will need a MacOS computer. 

1. Download Xcode
2. Open Xcode
3. Click "Clone Git Repository..."
4. Copy this Git repository link using this link: https://github.com/SirBabyWipes/weather-app.git
5. Paste it at the top of the new window, where it says "Search or enter repository URL"
6. Click "Clone"
7. If a window pops up telling you to choose a branch, select "main"
8. Now the project should clone and automatically open. At the top left, there should be a play button. Click that.
9. If at any point you are prompted to install something, do it -- you will need the emulator and any necessary SDKs in order to run the app.
10. A new window will open on your Mac, called "Simulator". Click on it and give the emulator and app a moment to start (it may be a white screen for a moment, or it may be installing something).
11. The weather should load and display on the screen!
